主成分提取(Principal Component Analysis problem formulation)

对于降维问题，目前最流行的，最常用的算法就是PCA主成分分析法。 PCA做的就是，寻找一条直线，或者面，或者诸如此类，比起特征向量低维的空间，对数据进行投影，并且最小化投影距离，也就是数据点和投影后的点之间的距离。另外，在寻找vectors时，对于所有的数据集和特征向量，都是同等对待的。

关于上图的左边，注意区别PCA和线性回归，两个截然不同的概念。

在使用PCA之前，我们通常会有一个数据预处理的过程，就是对数据进行均值归一化。
